# Face Detection App

A Computer Vision project developed using OpenCV to detect human faces in images using Haar Cascade classifiers.

## Project Overview

This project identifies and detects human faces from uploaded images and highlights them with bounding rectangles. The application demonstrates the fundamentals of image processing and face detection using Computer Vision techniques.

## Features

- Human face detection
- Image preprocessing
- Haar Cascade-based detection
- Face highlighting using bounding boxes
- Real-time image analysis

## Technologies Used

- Python
- OpenCV
- Matplotlib
- Computer Vision

## Concepts Implemented

- Image Processing
- Grayscale Conversion
- Haar Cascade Classifiers
- Object Detection

## Project Workflow

1. Upload image
2. Convert image to grayscale
3. Detect faces using Haar Cascade classifier
4. Draw rectangles around detected faces
5. Display processed image

## Output

- Detected faces highlighted with rectangles
- Total number of faces detected
